# Antigravity Project Rules

今後の Antigravity とのセッションでは、以下のルールに従って作業を進行します。

## 1. 言語設定
- ユーザーとの会話（チャットの返答）や、出力する各種アーティファクト（プラン、タスク、ウォークスルーなど）は**すべて日本語**で記述してください。
- （AIの内部的な思考プロセスについては英語でも構いません）

## 2. テストと動作確認
- テストを実行する際は以下のコマンドを使用してください。
  ```bash
  uv run pytest
  ```
- 実装タスクが完了した際は、**必ず上記のコマンドでテストを実行**し、既存の機能が壊れていないか、新規機能が正しく動作するかを検証してください。
