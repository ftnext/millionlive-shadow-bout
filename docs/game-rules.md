# 陰界戦戯 シャドウバウト ルール

<!-- https://x.com/imasml_theater/status/2047608333482537201 -->

## ゲームの目的
じゃんけん勝負を4回繰り返し、最終的に獲得した「勝ち札」のポイント合計数が高いプレイヤーが勝者となります。

## 準備 (01)
- **デッキ構築**: 同じカードは1枚まで（重複NG）。13種類のカードでデッキを作成します。
- **初期配置**: 自分のデッキをシャッフルして山札として置き、上から5枚を引いて手札とします。

## ゲームの進行

### 1. カードのセット (02)
相手に勝てそうなカードを手札から1枚選び、裏向きにして場にセットします。

### 2. オープンと勝敗判定 (03)
お互いにセットが完了したら、「エンゲージ！」の掛け声とともにカードをオープンします。
カードに記載された「じゃんけんマーク（グー・チョキ・パー）」によって勝敗を決定します。

### 3. 勝敗結果の処理
- **勝った場合 (04)**: 勝ったプレイヤーは、相手の場に出ているカードを自分の「勝ち札」置き場に移動します。自分が出した場のカードは「捨札」に移動します。

- **あいこの場合 (05, 06, 07)**:
  1. **効果の発動**: カードに書かれた「ポイント数」が**小さい方**から順番に効果を処理します。
  2. **再勝負**: お互いの効果処理が完了したら、効果による決着がつかない限り再度判定を行います。
  3. **ポイント勝負**: それでも「あいこ」のまま（じゃんけんの勝敗がつかない）であれば、ポイントの大きさを比べて**大きい方が勝ち**となります。
     *(※ポイントの計算例：手札が4枚の場合、「カードの基本ポイント(例:17) + 手札の枚数(4) = 21ポイント」となります)*
  4. **完全引き分け (07)**: 仮にポイント数も同じだった場合、場のカードを「あいこストック」に移動させます。
     - 次回以降の勝負で勝ったプレイヤーは、相手の場のカードだけでなく、相手の「あいこストック」のカードもまとめて勝ち札にできます。
     - その際、自分のあいこストックのカードは、場のカードと共に捨札になります。

### 4. ゲームの終了 (08)
この「じゃんけん勝負」を**4回**繰り返します。
最終的に「勝ち札」のポイント合計数を計算し、高かった方がゲームの勝者となります。

## 補足ルール（注意事項）
- カードを捨札に置く場合、もともとの持ち主側の捨札に置きます。
- カードを山札に戻す場合、もともとの持ち主の山札に戻します。
- あいこ かつ ポイントも同じ場合、効果は**カード名の五十音順が早い方**から発動します。
- マーク変更効果は、あいこ後の効果処理が終わった時点でじゃんけんを再判定します。
  - card_28（亜利沙）は、現在の勝負で相手の場のカードを即時に「チョキ」扱いにします。
  - card_09（律子）は、次の勝負だけ相手の場と手札のカードを「グー」扱いにし、その勝負が終わると効果は消えます。
- 「この勝負に勝つ」効果（card_31 朋花）は、成立した場合、効果後のじゃんけん再判定やポイント比較より優先して勝敗を上書きします。
  - 同じ勝負で複数の「この勝負に勝つ」効果が成立した場合は、効果処理順で後から成立した側を勝者とします。
- card_49（桃子）は、場に出すときに「グー」「チョキ」「パー」のいずれかを宣言し、この勝負では宣言したマークとして扱います。
- 手札がなくなるなどにより、場にカードを出せないプレイヤーは、その勝負は**不戦敗**となります。
  - 不戦敗となったプレイヤーは山札の上から1枚めくり、そのカードが相手の勝ち札となります。
  - (場にカードを出せるプレイヤーは必ず出さなければならず、不戦勝となった場合は出したカードを捨札に置きます。)
- お互いに場にカードを出せなくなった場合はゲームを終了し、その時点での勝ち札のポイント合計数で勝敗を決めます。
